How it works

Context optimized.
Not guessed.

01

Intercept

Redcon sits between your agent and the LLM as a drop-in gateway on localhost:8787. One env var change. No code to modify.

02

Score

Every file is ranked by import-graph proximity, recency, and BM25 similarity to the task string. Irrelevant files are dropped.

03

Compress

Off-path functions are elided. Docstrings stripped. Only the signal reaches the model — quality risk stays low.

04

Cache

Unchanged files are served from a per-session SHA-256 content cache. Warm-cache calls use 99% fewer tokens.
