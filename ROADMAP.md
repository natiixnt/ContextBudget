# ContextBudget Roadmap

## Near Term (v0.2)
- Better language-aware chunking (Python, TS, Go)
- Import graph signals in relevance scoring
- More robust token estimation backends
- Config file (`contextbudget.toml`)

## Mid Term (v0.3)
- Incremental scan index and file watcher mode
- Compare run-to-run drift and context deltas
- Plugin interface for alternate scorers/compressors

## Long Term (v1.0)
- Optional LLM-assisted summarization with strict fallback
- CI integration for budget policy enforcement
- Multi-repo workspace support
- Remote cache backends
